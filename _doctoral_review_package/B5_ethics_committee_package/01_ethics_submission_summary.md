# Ethics Committee Submission Package

## Project

Inclusive Learning Platform - ILP

## Purpose

Prepare formal ethical review before fieldwork and longitudinal dataset collection.

## Rule

No real student data collection before ethics approval, informed consent, assent for minors, and anonymization workflow validation.

## Scope

- Educational profile data
- Kolb, Felder-Silverman and Kuder assessment data
- Adaptive recommendation records
- Teacher validation records
- Future multimodal data only under separate approval
