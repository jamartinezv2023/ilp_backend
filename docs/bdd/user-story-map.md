# User Story Map

## Épica: Seguridad y acceso

### HU-AUTH-001 Inicio de sesión exitoso

Como usuario registrado,
quiero iniciar sesión con credenciales válidas,
para acceder a mis funcionalidades según mi rol.

### HU-AUTH-002 Credenciales inválidas

Como usuario,
quiero recibir una indicación cuando mis credenciales no sean válidas,
para corregir la información ingresada.

### HU-AUTH-003 Usuario bloqueado

Como usuario bloqueado,
quiero conocer que mi cuenta no permite acceso,
para solicitar apoyo institucional.

## Épica: Caracterización educativa

### HU-CHAR-001 Registrar caracterización del estudiante

Como docente,
quiero registrar información de caracterización de un estudiante,
para obtener un diagnóstico educativo inicial.

### HU-CHAR-002 Validar información requerida

Como docente,
quiero conocer qué información requerida falta,
para completar la caracterización del estudiante.

## Épica: Diagnóstico inclusivo

### HU-DIAG-001 Consultar diagnóstico educativo

Como docente,
quiero consultar el diagnóstico educativo de un estudiante,
para orientar decisiones pedagógicas.

### HU-DIAG-002 Identificar barreras de aprendizaje

Como orientador,
quiero revisar barreras identificadas en el diagnóstico,
para proponer apoyos pedagógicos.

## Épica: Recomendaciones DUA

### HU-DUA-001 Consultar recomendaciones pedagógicas

Como docente,
quiero consultar recomendaciones alineadas con DUA,
para ajustar mis estrategias de enseñanza.

## Épica: Eventos educativos

### HU-EVT-001 Registrar evento de evaluación completada

Como plataforma educativa,
quiero conservar un evento cuando una evaluación finaliza,
para permitir procesamiento asincrónico confiable.
