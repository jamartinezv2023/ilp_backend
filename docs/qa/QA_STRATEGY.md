# ILP QA Strategy

## Testing Pyramid

1. Unit tests
2. Component tests
3. Integration tests
4. API tests
5. BDD scenarios
6. E2E tests

## Patterns

- AAA Pattern
- BDD
- Gherkin
- Screenplay
- Cohn Test Pyramid

## Tools

- JUnit
- Mockito
- Testcontainers
- Serenity BDD
- Cucumber
- Playwright
- Vitest
- React Testing Library

## Quality Gates

- Build must pass
- Tests must pass
- Coverage must be collected
- SonarCloud must pass
- Critical vulnerabilities must be zero
