# ILP DORA Metrics Strategy

## Metrics

| Metric | Source |
|---|---|
| Deployment Frequency | CI/CD pipelines |
| Lead Time for Changes | Azure Boards + Pull Requests |
| Mean Time To Recovery | Bugs + Incidents |
| Change Failure Rate | Failed deployments + hotfixes |

## Governance

Every production-impacting issue must be linked to:

- Bug
- Incident
- Root Cause Analysis
- Remediation task
- Regression test
