# MVP-21A - Plan de Cierre Arquitectonico 2026-2027

## Objetivo

Construir la radiografia clase por clase, endpoint por endpoint y componente por componente para decidir:

- Que terminar.
- Que refactorizar.
- Que eliminar.
- Que congelar.
- Que automatizar despues con Screenplay + Serenity.
- Como desplegar con Docker, k3d y Helm.

## Regla principal

No se reestructura la arquitectura antes del trabajo de campo.

1. Cerrar backend al 95%.
2. Cerrar frontend al 100%.
3. Completar instrumentos.
4. Ejecutar piloto.
5. Capturar dataset institucional.
6. Posponer ML avanzado.
7. Automatizar E2E despues de estabilizar.
