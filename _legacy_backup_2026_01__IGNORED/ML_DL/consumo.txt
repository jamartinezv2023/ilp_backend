PARTE 4 — AUTOMATIZACIÓN

Puedes automatizar este script para ejecutarse cada día:

💠 En Windows PowerShell (tarea programada):
python "C:\ruta\export_dataset.py"

💠 O desde PostgreSQL con pg_cron:
SELECT cron.schedule(
  'daily_ml_export',
  '0 3 * * *',
  $$ REFRESH MATERIALIZED VIEW CONCURRENTLY ml.students_dataset_mv; $$
);
